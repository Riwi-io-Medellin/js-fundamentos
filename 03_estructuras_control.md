# Estructuras de Control Repetitivas

Las estructuras repetitivas permiten ejecutar un bloque de código múltiples veces.

## for

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

## while

```js
let contador = 0;
while (contador < 5) {
  console.log(contador);
  contador++;
}
```

## do...while

```js
let numero = 0;
do {
  console.log(numero);
  numero++;
} while (numero < 5);
```
