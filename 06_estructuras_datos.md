# Estructuras de Datos en JavaScript

## Arrays

```js
let frutas = ["manzana", "banana", "pera"];
console.log(frutas[0]); // manzana
```

## Objetos

```js
let persona = {
  nombre: "Ana",
  edad: 30
};
console.log(persona.nombre); // Ana
```
