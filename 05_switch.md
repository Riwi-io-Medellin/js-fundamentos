# Switch en JavaScript

`switch` permite manejar múltiples condiciones de forma limpia.

```js
const color = "verde";

switch (color) {
  case "rojo":
    console.log("El color es rojo");
    break;
  case "verde":
    console.log("El color es verde");
    break;
  default:
    console.log("Color no reconocido");
}
```
